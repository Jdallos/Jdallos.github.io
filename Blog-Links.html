Links

I've found everyone of these links invaluable, enjoy.


Free Code Camp

A great place to start learning to code. Accessible bite-sized challenges that go great with your morning coffee. Free Code Camp also have a great support community, a huge range of interesting articles and hands down the very best weekly emails I've ever recieved on any topic!


W3 schools

If you are here surely you are already familiar with W3 schools. It's your one-stop-shop for quick reference materials with a library of how to guides and challenges. Thank you W3.


Harvard's CS50

Not for the faint hearted, but if you have the time this is the course that really sparked my interest in coding. No prior knowledge required but don't underestimate the courses reccomended time commitment.
